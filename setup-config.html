Cấu Hình Supabase

Vui lòng cấu hình kết nối Supabase để sử dụng ứng dụng

1 Tạo tài khoản Supabase

Truy cập https://supabase.com và đăng ký tài khoản miễn phí.

Gói miễn phí (Free Tier) cho phép tạo 2 dự án, đủ dùng cho ứng dụng của bạn.

2 Tạo dự án mới

  1. Nhấn nút "New Project"
  2. Đặt tên dự án: QuanLyCongTrinh
  3. Đặt mật khẩu database (ghi nhớ mật khẩu này)
  4. Chọn khu vực gần nhất (VD: Southeast Asia)
  5. Nhấn "Create new project"
  6. Chờ khoảng 2-3 phút để dự án khởi tạo

3 Lấy thông tin kết nối

Sau khi dự án tạo xong, vào mục Project Settings → API:

Lấy từ mục Project URL
Lấy từ mục Project API keys → anon public

4 Tạo bảng dữ liệu

Vào mục SQL Editor, tạo query mới và chạy đoạn SQL sau:

-- =====================================================
-- TẠO BẢNG USERS (ĐĂNG NHẬP)
-- =====================================================
CREATE TABLE IF NOT EXISTS public.users (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    username TEXT NOT NULL UNIQUE,
    fullname TEXT,
    password TEXT NOT NULL,
    role TEXT NOT NULL DEFAULT 'user' CHECK (role IN ('admin', 'manager', 'user')),
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Tài khoản mặc định
INSERT INTO public.users (username, fullname, password, role) VALUES
('admin', 'Quản trị viên', 'admin123', 'admin'),
('manager', 'Quản lý', 'manager123', 'manager'),
('user', 'Người dùng', 'user123', 'user')
ON CONFLICT (username) DO NOTHING;

-- =====================================================
-- CÁC BẢNG NGHIỆP VỤ
-- =====================================================
CREATE TABLE IF NOT EXISTS projects (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name TEXT NOT NULL,
    place TEXT,
    start_date DATE,
    end_date DATE,
    notes TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS members (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name TEXT NOT NULL,
    type TEXT CHECK (type IN ('day', 'hour', 'hybrid', 'share')),
    rate NUMERIC,
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS jobs (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    project_id UUID REFERENCES projects(id) ON DELETE CASCADE,
    name TEXT NOT NULL,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS timesheets (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    date DATE NOT NULL,
    member_id UUID REFERENCES members(id),
    project_id UUID REFERENCES projects(id),
    job_id UUID REFERENCES jobs(id),
    value NUMERIC,
    notes TEXT,
    is_settled BOOLEAN DEFAULT false,
    closure_id UUID,
    settlement_date TIMESTAMPTZ,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS cash (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    date DATE NOT NULL,
    project_id UUID REFERENCES projects(id),
    type TEXT CHECK (type IN ('in', 'out', 'advance')),
    amount NUMERIC,
    notes TEXT,
    member_id UUID REFERENCES members(id),
    payee_id UUID REFERENCES members(id),
    is_settled BOOLEAN DEFAULT false,
    closure_id UUID,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS closures (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    project_id UUID REFERENCES projects(id),
    all_project_ids JSONB,
    member_id UUID REFERENCES members(id),
    date DATE,
    amount NUMERIC,
    debt_left NUMERIC,
    notes TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS logs (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    timestamp TIMESTAMPTZ DEFAULT NOW(),
    user_id TEXT,
    user_name TEXT,
    action TEXT,
    entity_type TEXT,
    entity_id TEXT,
    details JSONB
);

-- TẮT RLS (Row Level Security) để đơn giản hóa
ALTER TABLE users DISABLE ROW LEVEL SECURITY;
ALTER TABLE projects DISABLE ROW LEVEL SECURITY;
ALTER TABLE members DISABLE ROW LEVEL SECURITY;
ALTER TABLE jobs DISABLE ROW LEVEL SECURITY;
ALTER TABLE timesheets DISABLE ROW LEVEL SECURITY;
ALTER TABLE cash DISABLE ROW LEVEL SECURITY;
ALTER TABLE closures DISABLE ROW LEVEL SECURITY;
ALTER TABLE logs DISABLE ROW LEVEL SECURITY;
Quan trọng: Sau khi chạy SQL thành công, bạn sẽ thấy danh sách các bảng được tạo.
